# ✨ Overview

TickrPay is a modern payment and attendee management platform designed for events, registrations, ticketing, and real-time payment tracking.

The platform combines:

* Fast event registration
* Secure attendee payments
* Real-time dashboards
* Smart validation systems
* Duplicate prevention logic
* Expandable multi-event architecture

Built using the latest Next.js App Router architecture and powered by Supabase for authentication, database services, and real-time synchronization.

# 📂 Project Structure

```bash
tickrpay/
│
├── app/                # Next.js App Router
├── components/         # Shared reusable UI components
├── lib/                # Core utilities & business logic
├── supabase/           # Supabase configuration & helpers
├── types/              # Global TypeScript definitions
├── public/             # Static assets
├── styles/             # Global styles
│
├── .env.local
├── package.json
└── README.md
```

---

# ⚙️ Environment Setup

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

# 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:3000
```
